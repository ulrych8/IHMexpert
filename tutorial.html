Welcome to our study
Introduction: You are invited to participate in a study to improve understanding of user
behavior with an interactive system.
            
This study is conducted by Gilles Bailly and Evanthia Dimara (Sorbonne University) who will
be delighted to answer your questions ( gilles.bailly@sorbonne-universite.fr ). This study
was approved by The INSEAD Internal Review Board (reference 201915).
            
Task: The task is to select symbols as quickly as possible.
You will first make a tutorial in three steps to learn how to select symbols:
    - Tuto 1: Learn to use the Novice method.
    - Tuto 2: Learn to use the Expert method.
    - Tuto 3: Learn to unlock the Expert method.
            
Press the "Start Tutorial" button to begin the tutorial.
TUTO 1
Learn to use the Novice method.
            
1. Select in the grid all the symbols corresponding to the target
symbol.

2. Once all target symbols are selected, the “Next” button will be clickable.

3. Press the button to continue.

4. Repeat this step three times.
TUTO 2
We will now use the expert method that is 3 times faster.
            
1. Select the target symbol or one of the symbol corresponding to the target symbol in the grid. 
Automatically, all similar symbols in the grid will be selected.

2. Once all target symbols are selected, the Next button will be clickable.

3. Press the button to continue.

4. Repeat this task three times.
TUTO 3
We will now learn how to unlock the expert method.
Initially, the expert method associated with each symbol is locked by several lockers.
You need to unlock ALL the lockers to be able to use the expert method.
But each time, you can only open one locker at the time.
            
Follow these steps to unlock the expert method :
                
1. Select all the forms corresponding to the target form.
            
2. Here, instead of clicking on the next button, click on the unlock button 
underneath the target symbol.
3. The slider that just appear need to be 'slided' to unlock one padlock.
So slide the slider until it disappear and a padlock turn white.
                
4. Now click on the next button and reiterate that process one more time.
                
5. Use the expert mode for this form.
